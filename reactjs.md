create react app
```cmd
    npx create-react-app app_name
```

create react app vite
```cmd
    npm create vite@latest

    project name: app_name
    select a framework: select_react
    select a variant: javascript

    cd app_name
    npm install
```

run app
```cmd
    npm start
    npm run start
    npm run dev
    npm run debug
```
